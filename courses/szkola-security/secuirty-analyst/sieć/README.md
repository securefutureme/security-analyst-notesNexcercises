# Sieć

Sekcja zawiera notatki oraz ćwiczenia praktyczne dotyczące podstaw sieci komputerowych, analizy ruchu oraz monitorowania bezpieczeństwa sieciowego.

Materiały koncentrują się na umiejętnościach przydatnych w pracy analityka SOC, takich jak analiza plików PCAP, identyfikacja anomalii oraz wykrywanie ataków sieciowych.

## Zakres materiału

* podstawy komunikacji TCP/IP,
* protokoły i porty sieciowe,
* analiza ruchu w Wiresharku,
* analiza plików PCAP i PCAPNG,
* Network Security Monitoring,
* systemy IDS,
* Snort i Zeek,
* wykrywanie skanowania sieci,
* wykrywanie ataków DDoS,
* wykrywanie ataków Man-in-the-Middle,
* identyfikacja adresów IP, domen i innych IOC,
* tworzenie osi czasu aktywności sieciowej.
