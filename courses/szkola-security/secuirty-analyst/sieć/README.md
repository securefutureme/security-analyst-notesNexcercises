# Sieć

Sekcja zawiera notatki oraz ćwiczenia praktyczne dotyczące podstaw sieci komputerowych, analizy ruchu oraz monitorowania bezpieczeństwa sieciowego.

Materiały koncentrują się na umiejętnościach przydatnych w pracy analityka SOC, takich jak analiza plików PCAP, identyfikacja anomalii oraz wykrywanie ataków sieciowych.
