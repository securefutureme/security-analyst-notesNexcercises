# Network Security Monitoring

Sekcja zawiera notatki dotyczące **Network Security Monitoring (NSM)**, czyli monitorowania ruchu sieciowego w celu wykrywania podejrzanej aktywności, prób ataków oraz naruszeń bezpieczeństwa.

Materiały obejmują podstawy NSM, systemy IDS, analizę ruchu, wykorzystanie Snorta oraz przykładową analizę incydentu.

## Główne zagadnienia

- analiza ruchu sieciowego,
- identyfikacja anomalii,
- systemy IDS,
- analiza alertów,
- wykrywanie ataków MitM,
- wykrywanie eksfiltracji danych,
- analiza incydentów sieciowych,
- wykorzystanie Snorta w monitoringu bezpieczeństwa.
